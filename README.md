# Alfred 2 Workflows

## Real Download Link

获取国内一些下载软件所特有链接的真实下载地址，支持迅雷、快车、QQ旋风，也支持ed2k、http、ftp链接的一些基本信息的解析。

## Chinese Dictionary

基于[QQ词典][]的中英互译。

* ENTER 打开QQ词典页面
* CTRl + ENTER 打开爱词霸页面
* OPTION + ENTER 发音
